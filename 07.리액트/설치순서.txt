* npm 이란?n : 노드(node) / p : 패키지(package) / m : 매니저(managemer)- 
npm은 node.js 를 설치시 함께 설치됨!- 

node.js를 사용하여 만들어진 소프트웨어이지만 
타 소프트웨어를 사용할 수 있는 모듈! 소프트웨어 관리자 같은

★리액트를 사용하기 위해서는 기본적으로 
Node.js가 설치되어 있어야 한다.

1.cmd로 명령 프롬프트 접속

2. cd/ 입력해서 C:\로 경로 변경

3. npm install -g create-react-app            
   (//권한 없다고 오류 뜰수도 있음. -g는 global 컴퓨터 어디에서도 실행 할 수 있게 한다는 뜻.)

4. npx create-react-app my-app(폴더이름)   
5. cd my-app 로 폴더 이동
6. npm start
